# Password Strength Indicator

A simple tool to check password strength in real-time. Perfect for anyone learning JavaScript and UI interactions.

## Live Demo
- GitHub Pages: [https://sheebamobin.github.io/Password-strength-indicator/](https://sheebamobin.github.io/Password-strength-indicator/)
- Vercel: [password-strength-indicator-brown.vercel.app](password-strength-indicator-brown.vercel.app)


## How it Works
Type a password and see if it's weak, medium, or strong instantly.
